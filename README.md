# Unit Converter

Instant unit conversion for common measurements.

## Categories

- **Length** - meters, feet, inches, miles, km
- **Weight** - kg, pounds, ounces, grams
- **Temperature** - Celsius, Fahrenheit, Kelvin
- **Volume** - liters, gallons, cups, ml
- **Area** - sq meters, sq feet, acres
- **Speed** - mph, km/h, m/s, knots
- **Data** - bytes, KB, MB, GB, TB
- **Time** - seconds, minutes, hours, days

## Features

- **Instant Conversion** - Results as you type
- **Reverse Conversion** - Swap units easily
- **Copy Results** - One-click copy
- **Common Presets** - Quick access to popular conversions

## Usage

1. Open `index.html` in a browser
2. Select category
3. Enter value
4. Choose source and target units
5. See result instantly

## Tech

- Pure HTML/CSS/JavaScript
- No external APIs
- Works offline
- Precise decimal handling
